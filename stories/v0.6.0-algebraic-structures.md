# Story: v0.6.0 代数结构基础

> 实现 Semigroup、Functor、Alternative 代数结构

## 目标

实现函数式编程的核心代数结构：Semigroup（结合操作）、Functor（映射操作）、Alternative（选择操作）。

- [x] `intersperse` - 插入分隔符
